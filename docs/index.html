What it does

Everything you need, automated

From bare Windows Server to a fully hardened Exchange installation — 6 phases, fully unattended.

🔁

AutoPilot — Zero-Touch Install

Automatically reboots and resumes at the correct phase via RunOnce. State is persisted to XML — safe against crashes and power failures.

🛡

Security Hardening

TLS 1.2/1.3, SMBv1 disabled, WDigest off, LSA Protection, Serialized Data Signing, NTLM level 5, Credential Guard, HTTP/2 disabled — all applied automatically.

⚡

Performance Tuning

High Performance power plan, pagefile sizing, TCP settings, RSS + NIC queues, Netlogon MaxConcurrentAPI, Exchange Search affinity — per Microsoft & CSS-Exchange recommendations.

📋

HTML + PDF Installation Report

Comprehensive post-install report: Exchange config, virtual directory URLs, certificates, security settings with best-practice comparison, HealthChecker results embedded.

🖥

Interactive Menu

Start without parameters for a guided installation menu. Instant toggle switches, credential validation, auto-detected config.psd1, and PS2Exe-compiled .exe support.

🔄

Swing Migration Ready

Export config from source server, import virtual directory URLs and connectors, import PFX certificate, join a Database Availability Group — all in one run.

🔒

Certificate Management

Import PFX certificate, enable for IIS + SMTP (+ IMAP/POP for non-wildcard), configure HSTS on OWA/ECP. Wildcard vs. named cert detection.

🧰

CSS-Exchange Integration

HealthChecker auto-downloaded and run in Phase 6. SetupAssist triggered on Phase 4 failure. EOMT for CVE mitigation. Auth Certificate monitoring setup.

📦

.exe Compilation

Build.ps1 wraps the script into a self-contained Windows executable via PS2Exe — runs elevated, preserves all parameters, includes version metadata.

How it works

6 phases, one command

State persists in an XML file — resume automatically after every reboot.

0

Preflight & HTML Report

Admin rights, domain membership, OS version, AD Forest/Domain level, static IP, Exchange version, free disk space. Pre-flight HTML report generated.

1

Windows Features & Updates

All required Windows Server roles and features installed in a single batch. Optional Windows security updates. Source server config exported.

2

Prerequisites

.NET Framework 4.8 / 4.8.1, OS-specific hotfixes, Visual C++ 2012 / 2013 Redistributables, URL Rewrite Module 2.1.

3

UCMA & Active Directory Preparation

Unified Communications Managed API 4.0. PrepareAD / PrepareSchema / PrepareAllDomains with optional AD replication wait.

4

Exchange Setup

Runs setup.exe with the correct role switches. Autodiscover SCP pre-configured. Transport services set to Manual for safe startup. SetupAssist on failure.

5

Post-Configuration

28 security and performance tasks: TLS, Schannel, Defender exclusions, power plan, pagefile, NIC RSS, LSA Protection, certificate import, HSTS, Exchange SU, virtual directory URLs.

6

Finalization

Start services, IIS health check, DAG join, RBAC report, anti-spam agents, send connector integration, log cleanup task, HealthChecker, HTML + PDF installation report.

Quick Start

Up and running in minutes

Interactive menu or fully unattended — your choice.

# Interactive menu (recommended for first run) .\Install-Exchange15.ps1 # Fully unattended — AutoPilot handles reboots automatically .\Install-Exchange15.ps1 -SourcePath D:\Exchange -AutoPilot # Load all parameters from a config file .\Install-Exchange15.ps1 -ConfigFile .\deploy-mbx01.psd1 # Swing migration: copy config, import certificate, join DAG .\Install-Exchange15.ps1 -SourcePath D:\Exchange -AutoPilot ` -CopyServerConfig EX01 -CertificatePath D:\certs\mail.pfx -DAGName DAG01 # Run all post-install optimizations on an existing Exchange server .\Install-Exchange15.ps1 -StandaloneOptimize -Namespace mail.contoso.com ` -CertificatePath C:\certs\mail.pfx -LogRetentionDays 30 # Compile to self-contained .exe .\Build.ps1
